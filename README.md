# 💻 Laptop Price Predictor

A Machine Learning powered web app that predicts the price of a laptop based on its specifications. Built using Python, Scikit-learn, and Streamlit.

---

## 🚀 Live Demo

👉 https://subhit-coder-laptop-price-prediction-app-ejyexo.streamlit.app/

---

## 📌 Features

* 🔍 Predict laptop price instantly
* 💻 Supports multiple brands and configurations
* 📊 Uses trained ML pipeline model
* 🎯 Clean and interactive UI with Streamlit
* ⚡ Fast and accurate predictions

---

## 🧠 Machine Learning

* Algorithm Used: Regression Model
* Pipeline: Data Preprocessing + Model
* Features Used:

  * Brand
  * Type
  * RAM
  * Weight
  * Touchscreen
  * IPS Display
  * Screen Size & Resolution (PPI)
  * CPU
  * HDD / SSD
  * GPU
  * Operating System

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## ⚙️ Installation & Setup

1️⃣ Clone the repository

```
git clone https://github.com/your-username/laptop-price-predictor.git
cd laptop-price-predictor
```

2️⃣ Install dependencies

```
pip install -r requirements.txt
```

3️⃣ Run the app

```
streamlit run app.py
```
